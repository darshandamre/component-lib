Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Image avatars

Image avatars can be created by giving the class name as avatar to an image. 

<img class="avatar" src="https://url-of-image.jpg" alt="avatar" />

Sizes

Avatars are availbale in 3 different sizes. The default size is 3x3 rem (48x48 px). For smaller avatars, add class avatar--sm (2x2 rem or 32x32 px). For larger avatars, add class avatar--lg (4.5x4.5 rem or 72x72px) 

<img class="avatar avatar--lg" src="https://url-of-image1.jpg" alt="avatar" />
              <img class="avatar" src="https://url-of-image2.jpg" alt="avatar" />
              <img class="avatar avatar--sm" src="https://url-of-image3.jpg" alt="avatar" />

Letter avatars

Letter avatars can be created by giving avatar avatar--text classes to a div and having 1 or 2 letters inside that div. Large and small letter avatars can be created by adding class avatar--lg and avatar--sm respectively

VM
TP
L 
<div class="avatar avatar--text avatar--lg" role="img" alt="avatar" >VM</div>
              <div class="avatar avatar--text" role="img" alt="avatar" >TP</div>
              <div class="avatar avatar--text avatar--sm" role="img" alt="avatar" >L</div>

Button

There are several predefined button styles for different purposes.

Primary and secondary buttons

Use class btn for buttons and then btn--primary or btn--secondary for primary or secondary colored buttons. For pill shaped button, use class btn--pill. 

<button class="btn btn--primary">Primary</button>
              <button class="btn btn--primary btn--pill">Primary</button>
              <button class="btn btn--primary" disabled>Primary</button>
              
              <button class="btn btn--secondary">Secondary</button>
              <button class="btn btn--secondary btn--pill">Secondary</button>
              <button class="btn btn--secondary" disabled>Secondary</button>

Link buttons

Use classes btn btn--link for links.

Link 
<a href="#" class="btn btn--link">Link</a>

Icon buttons

Use classes btn btn--link for icon buttons. 

<button class="btn btn--icon">
                 <i class="far fa-envelope fa-2x"></i>
              </button>

              <button class="btn btn--icon">
                 <i class="far fa-bookmark fa-2x"></i>
              </button>

Floating action button

Use classes btn btn--link for floating action button. 

<a href="#" class="btn btn--floating">
                 <i class="fas fa-arrow-circle-up fa-2x"></i>
              </a>

Images

Responsive images

For responsive images, use class image-resp on img tags. 

<img class="image-resp" src="../assets/memento-mori-playing-cards.jpg" alt="memento mori playing cards" />

Round images

For round images, use class image-round in a div and put img tag inside it. 

<div class="image-round">
               <img src="../assets/memento-mori-playing-cards.jpg" alt="memento mori playing cards" />
            </div>

Input

Use the following code snippet to use these input components. 

<label for="input">Label</label>
              <input class="input" id="input" type="text" placeholder="Placeholder" />
              
              <label for="input--error">Label</label>
              <div class="input__container--error">
                 <input class="input input--error" id="input--error" type="text" placeholder="Placeholder" />
                 <i class="error-icon fas fa-exclamation-triangle"></i>
              </div>
