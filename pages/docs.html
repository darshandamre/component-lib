Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Image avatars

Image avatars can be created by giving the class name as avatar to an image.

<img class="avatar" src="https://url-of-image.jpg" alt="avatar" />

Sizes

Avatars are availbale in 3 different sizes. The default size is 3x3 rem (48x48 px). For smaller avatars, add class avatar--sm (2x2 rem or 32x32 px). For larger avatars, add class avatar--lg (4.5x4.5 rem or 72x72px)

<img class="avatar avatar--lg" src="https://url-of-image1.jpg" alt="avatar" />
<img class="avatar" src="https://url-of-image2.jpg" alt="avatar" />
<img class="avatar avatar--sm" src="https://url-of-image3.jpg" alt="avatar" />

Letter avatars

Letter avatars can be created by giving avatar avatar--text classes to a div and having 1 or 2 letters inside that div. Large and small letter avatars can be created by adding class avatar--lg and avatar--sm respectively

VM
TP
L
<div class="avatar avatar--text avatar--lg" role="img" alt="avatar" >VM</div>
<div class="avatar avatar--text" role="img" alt="avatar" >TP</div>
<div class="avatar avatar--text avatar--sm" role="img" alt="avatar" >L</div>

Badge

Badges are used to display status or count information to the user.

Badge on Avatars

Status badges can be created by the following way. The default color is grey, you change the color to red, yellow or green by using the examples given below.

<div class="badge-container">
  <img class="avatar" src="https://url-of-image1.jpg" alt="avatar" />
  <span class="badge" ></span>
</div>
<div class="badge-container">
  <img class="avatar" src="https://url-of-image2.jpg" alt="avatar" />
  <span class="badge badge--red" ></span>
</div>
<div class="badge-container">
  <img class="avatar" src="https://url-of-image3.jpg" alt="avatar" />
  <span class="badge badge--yellow" ></span>
</div>
<div class="badge-container">
  <img class="avatar" src="https://url-of-image3.jpg" alt="avatar" />
  <span class="badge badge--green" ></span>
</div>

Badge on Icons

Badge on icons can be used to show the number of messages or items in cart. The above colors can be used with badge on icons in the same way. Default backgound color is blue.

4
99+

  <span class="badge-on-icon badge--green" >4</span>
</div>
<div class="badge-container">

</div>
